## What
Backend interview prep using roadmap.sh backend roadmap (backend.pdf in C:\IBM\Learning\BE\).
CLAUDE.md contains all teaching and session instructions.
tracker.md tracks progress across sessions.

## Rules (summary — full detail in CLAUDE.md)
- Teach topic by topic, update tracker.md after each subtopic
- Create notes.md per subtopic in `TopicNumber-TopicName/SubtopicNumber-Name/`
- Notes are revision-ready: bullets, tables, no paragraphs
- Always do interactive Q&A after each subtopic with structured feedback
- Include interview questions at end of every subtopic
- User can jump ahead to any topic on request

## Why
User doesn't want to re-type instructions each session. CLAUDE.md + tracker.md carry full context forward.

## How to apply
At session start: read tracker.md → resume from last in-progress topic. All behavior rules are in CLAUDE.md.
